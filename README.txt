# DiscoverMyPotential

Brief Discription:
================== 
DiscoverMyPotential is a website that makes indiviuals and small entreprises to show their potientials, skills, and other things. It serves as a portfolio for the user. 
The current version of the website is the very first one.

Use Cases:
==========
User can Choose which category of portfolio(personal, technical,resume, CV, services , products) and each category have multiple types, user must choose the ones that suits his needs.
User can pick up only one portfolio.
User can Contact the admin if he needs a new field through a special form (not the through website contact form).
User can link his portfolio with other social media and website (faceBook , LinkdIn, twitter, G+, github, Personal Website).
User only can modify some colors of his portfolio.
